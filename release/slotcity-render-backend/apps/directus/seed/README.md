# Directus Seed

Use these files as starter import payloads for the Directus content layer.

Current files:

- `route-payloads.json`
- generated by `pnpm --filter directus seed:route-payloads`
- push to Directus with `pnpm --filter directus seed:push-route-payloads`

Expected collection:

- `storefront_route_payloads`

Suggested import workflow:

1. Create the `storefront_route_payloads` collection with fields `slug`, `payload`, `status`.
2. Import `route-payloads.json` or paste each `payload` into the matching item.
3. Set `DIRECTUS_URL` and `STOREFRONT_CONTENT_MODE=directus` for `apps/web`.
4. Publish the items, then verify `home`, `catalog`, and `live` against Chrome control screenshots.

API upsert:

- `DIRECTUS_URL=https://cms.slotcity.ua DIRECTUS_TOKEN=... pnpm --filter directus seed:push-route-payloads`
- `DIRECTUS_DRY_RUN=true pnpm --filter directus seed:push-route-payloads`
