# Environment Reference

Use this file as the copy/paste source for production setup.

## Render: `slotcity-directus`

```env
PORT=8055
PUBLIC_URL=https://cms.slotcity.ua
SECRET=generate-in-render
KEY=generate-in-render
ADMIN_EMAIL=admin@slotcity.ua
ADMIN_PASSWORD=replace-with-strong-password
DB_CLIENT=pg
DB_HOST=replace-with-postgres-host
DB_PORT=5432
DB_DATABASE=slotcity
DB_USER=replace-with-postgres-user
DB_PASSWORD=replace-with-postgres-password
REDIS=redis://replace-with-redis-host:6379
CORS_ENABLED=true
CORS_ORIGIN=https://slotcity.ua
STORAGE_LOCATIONS=s3
STORAGE_S3_DRIVER=s3
STORAGE_S3_BUCKET=slotcity-assets
STORAGE_S3_REGION=auto
STORAGE_S3_ENDPOINT=replace-with-r2-or-s3-endpoint
STORAGE_S3_KEY=replace-with-storage-key
STORAGE_S3_SECRET=replace-with-storage-secret
```

Notes:
- `SECRET` and `KEY` should be generated by Render.
- `PUBLIC_URL` should be the public CMS domain that Cloudflare routes to Render.
- if you use Cloudflare R2, keep `STORAGE_S3_REGION=auto`.

## Vercel: `apps/web`

```env
NEXT_PUBLIC_SITE_URL=https://slotcity.ua
NEXT_PUBLIC_POSTHOG_KEY=replace-with-posthog-project-key
NEXT_PUBLIC_POSTHOG_HOST=https://us.i.posthog.com
NEXT_PUBLIC_SENTRY_DSN=replace-with-sentry-dsn
NEXT_PUBLIC_ENABLE_SESSION_REPLAY=true
DIRECTUS_URL=https://slotcity.ua/cms
EVENTS_API_URL=https://slotcity.ua/events
SENTRY_ENVIRONMENT=production
OTEL_SERVICE_NAME=slotcity-web
STOREFRONT_CONTENT_MODE=directus
```

Notes:
- `DIRECTUS_URL` and `EVENTS_API_URL` should point to the public Cloudflare-routed URLs, not raw Render URLs.
- `NEXT_PUBLIC_POSTHOG_KEY` is the browser project key.
- `NEXT_PUBLIC_SENTRY_DSN` is the frontend DSN for `apps/web`.

## Cloudflare Worker vars

```env
WEB_ORIGIN=https://slotcity-web.vercel.app
DIRECTUS_ORIGIN=https://slotcity-directus.onrender.com
EVENTS_ORIGIN=https://slotcity-events.onrender.com
```

Notes:
- `WEB_ORIGIN` is the Vercel deployment domain.
- `DIRECTUS_ORIGIN` is the Render Directus origin.
- `EVENTS_ORIGIN` is the Render events origin.

## Directus push commands

Generate route payloads:

```bash
corepack pnpm --filter directus seed:route-payloads
```

Dry run:

```bash
DIRECTUS_DRY_RUN=true corepack pnpm --filter directus seed:push-route-payloads
```

Push to Directus:

```bash
DIRECTUS_URL=https://cms.slotcity.ua \
DIRECTUS_TOKEN=replace-with-directus-static-token \
corepack pnpm --filter directus seed:push-route-payloads
```
